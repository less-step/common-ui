### Less-step 组件库
#### 安装方式

```
    npm install less-step-common
```
#### 支持组件有
    * 按钮组件 Button
    * 输入框组件 Input
    * 自动补全组件 AutoComplete
    * 菜单组件 Menu
    * 百分比组件 Percentage
    * 上传组件 Upload
    * 表单组件 Form
    * 动效组件 Transition
